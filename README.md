# KnowBase

[![Live Demo](https://img.shields.io/badge/Live-Demo-green?style=for-the-badge)](https://knowbase.rf.gd)

Unified control room to build, launch, and govern AI assistants.  
Manage documents, deploy chat widgets, monitor usage, and resolve unanswered conversations — all from one workspace.

## 🚀 Key Features

### 🤖 Bots & Chat
- Create assistants
- Theme per-bot widgets
- Embed via a single `<script>` tag

### 📄 Document Ingest
- Upload and scope content per bot
- Built-in limits and quotas

### 👀 Live Preview
- See widget changes instantly
- Mobile-friendly floating launcher

### 📊 Analytics
- Track sessions
- Monitor unanswered conversation spikes
- Track quota consumption in real time

### 🛡 Governance
- Domain guardrails
- API key rotation
- Audit-friendly workflows

### ⚙ Operations
- Health checks
- Safe database handling for pooled Postgres (Supabase)
- Autocommit pooling for pooler setups

---

## ✅ Why Use It

### ⚡ Faster Shipping
Live previews and one-click embeds reduce integration time.

### 🔐 Safer at Scale
Guardrails, quotas, and per-bot isolation reduce blast radius.

### 📈 Operational Clarity
Real-time metrics surface issues before they impact CSAT.

### 🌍 Flexible Deployment
- Backend ready for Render + Supabase
- Widget works on any site with one script tag
